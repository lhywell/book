Leaflet.js
====================

@(笔记)[leaflet.js]

-------------------

开源的地图交互的Javascript库,支持桌面、移动端，官网：http://leafletjs.com/

1. 大小38K
2. 插件
3. 支持链式调用
```js
var map = L.map('mapid').setView([51.505, -0.09], 13);
```
4. L.tileLayer支持各种底层
